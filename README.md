# Animal Tables

A simple web application that displays tables of animals from different categories, such as big cats, dogs, and big fish. The app allows users to view animal details, sort them by different attributes, edit animal information, and delete animals from the table. It also supports image previews with hover effects.

## Features

- **Animal Categories**: Displays tables of animals in categories like Big Cats, Dogs, and Big Fish.
- **Sorting**: Sort the table based on the name, location, or size of the animals.
- **Edit**: Edit the name of any animal in the table.
- **Delete**: Remove any animal from the table.
- **Image Hover Effect**: Hover over animal images to see their name.
